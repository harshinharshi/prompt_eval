How It Works

1. Input Configuration

Enter your evaluation goal and the system prompt you want to test.

2. Test Case Generation

The system generates test cases or uses your custom ones in JSON format.

3. Response Generation

Selected LLM responds to each test case with the provided system prompt.

4. Automated Grading

A grading LLM evaluates each response based on your goal (1-10 scale).

5. Results Display

View individual grades, reasoning, and average score.
